# Masiha
Flood management Assam

# Getting Started with Our website

# cd website
Go to the website folder inside main app

# npm install
Install npm dependencies
# cd client 
change the directory to client app
# npm install 
Again install npm dependencies
# cd ..
Change directory
# npm run dev
Run the server
